얼굴인식 (Face Recongnition)

아래에 주어진 예시 사진을 활용하여 허스키렌즈로 학습시켜보세요.
배우지 않은 것을 배울 준비가 되었다면 화면에 흰색 Face 프레임이 나타납니다.
버튼을 눌러 학습시켰다면 파란색 프레임으로 바뀌고 'Face : ID 1'이라고 나타납니다!

만약 여러 사람의 얼굴을 한 번에 인식시키고 싶다면
다이얼(Function) 버튼을 2초간 누르기 -> Learn Multiple로 가서 다이얼 버튼을 눌러 선택하고 좌,우로 돌려 조절하기 --
-> Save & Return로 가서 다이얼버튼을 누르기 -> 'Do you save data?' 질문에 'Yes'로 다이얼 버튼을 누르기
(오른쪽으로 갈수록 인식 양이 많아짐)







오브젝트추적 (Object Tracking)

이것은 허스키렌즈가 물체를 추적하는 기능입니다.
여러분의 필기구나 물건을 활용하여 허스키렌즈로 학습시켜보세요.
배우지 않은 것을 배울 준비가 되었다면 화면에노란색 프레임이 나타납니다.
버튼을 눌러 학습시켰다면 노란색 프레임 위에 'Learning : ID 1'이라고 나타납니다!







오브젝트인식 (Object Recognintion)

이것은 허스키렌즈가 기본적으로 학습되어있는 물체를 제공하는 기능입니다.
물병(Bottle), 자동차(Car), 버스(Bus), 고양이(Cat), 강아지(Dog), 새(Bird) 등을 인식합니다.
한번 인식시켜 보세요!







라인추적 (Line Tracking)

이것은 허스키렌즈가 선을 따라 화살표 모양으로 추적합니다.
배우지 않은 것을 배울 준비가 되었다면 화면에 노란 십자가와 흰색 화살표가 나타납니다.
초점을 맞추고 버튼을 눌러 학습시켰다면 파란색 화살표로 나타납니다.
아래의 예시 사진과 주변을 둘러보며 선을 찾아 추적해보세요!







색상인식 (Color Recognintion)

아래에 주어진 예시 사진을 활용하여 허스키렌즈로 학습시켜보세요.
배우지 않은 것을 배울 준비가 되었다면 화면에 노란 십자가와 흰색 프레임'이 나타납니다.

태그인식 (Tag Recognintion)

아래에 주어진 예시 사진을 활용하여 허스키렌즈로 학습시켜보세요.
학습시키길 원하는 태그에 주황색 십자가를 정확히 맞추어 학습시켜주세요.
버튼을 눌러 학습시켰다면 해당 태그를 인식시켰을 때, 파란색 프레임으로 바뀌며 'Tag : ID 1'이라고 나타납니다!

(노란색 프레임이 화면에 나타난다면 허스키렌즈가 학습 중임을 나타냅니다.)

만약 여러개의 태그를을 한 번에 인식시키고 싶다면
다이얼(Function) 버튼을 2초간 누르기 -> Learn Multiple로 가서 다이얼 버튼을 눌러 선택하고 좌,우로 돌려 조절하기 --
-> Save & Return로 가서 다이얼버튼을 누르기 -> 'Do you save data?' 질문에 'Yes'로 다이얼 버튼을 누르기
(오른쪽으로 갈수록 인식 양이 많아짐)
